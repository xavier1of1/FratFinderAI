# Documentation

This folder keeps long-form project documentation out of the repository root.

- `NEW_DEVELOPER_GUIDE.md`: high-level onboarding guide for new engineers joining the project.
- `plans/`: implementation plans and execution roadmaps.
- `Diagrams/`: Mermaid-based architecture visuals and execution-flow diagrams.
- `reports/`: validation reports, project reports, and audit outputs.
