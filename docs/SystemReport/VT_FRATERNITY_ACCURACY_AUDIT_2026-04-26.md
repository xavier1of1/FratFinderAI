## Update Addendum (2026-04-27)

This section preserves the original audit exactly as written above and records the post-audit changes plus the latest live Virginia Tech counts.

### Changes Made After The Original Audit

- Confirmed intake sources now persist refreshed source-quality metadata instead of keeping stale weak discovery state.
- Official national directory URLs now score more accurately across:
  - web intake source evaluation
  - crawler discovery source evaluation
  - request-graph recovery evaluation
- Operator-confirmed official directory sources are now preserved on zero-record crawls instead of being automatically pushed back to `awaiting_confirmation` when no stronger alternate source exists.
- Chapter normalization now infers school identity from stronger context, including:
  - chapter names such as `Virginia Tech Provisional Chapter`
  - snippets such as `... at Virginia Tech`
  - school-coded source URLs such as `virginiatech.<chapter-domain>`
- Official provisional chapter rows with institution signal can now auto-promote into `chapters` even before website/email/Instagram enrichment lands.

### Data-Backed Impact Of The Code Changes

- In the original VT audit, `17` cases were in `source_guard_reverted_to_confirmation`.
- Re-evaluating those same `17` source URLs after the source-quality and preservation changes shows:
  - `13 / 17` now qualify as strong confirmed official-directory sources
  - `13 / 17` are now preservable on zero-record recovery instead of being forced back to `awaiting_confirmation`
- Before the refinement, only `2 / 17` met that preservation bar.
